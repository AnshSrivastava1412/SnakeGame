# Snake Game

A simple console-based Snake game written in C++ using linked lists to model the snake body.  
Run in Windows terminal, controlled with W (up), A (left), S (down), and D (right) keys.

## How to Build and Run

1. Compile with g++:

2. Run the compiled executable:

- On Windows:

  ```
  SnakeGame.exe
  ```

3. Control the snake using W/A/S/D keys. Avoid running into walls or yourself!
